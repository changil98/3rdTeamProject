## 🖥️ 프로젝트 소개

**스파르타코딩클럽 내일배움캠프 팀프로젝트**
 
추억의 게임 똥피하기를 현대적인 버전으로 재현한 게임

## 🕰️ 개발 기간
24.5.16 ~ 24.5.23

## 🧑‍🤝‍🧑 팀원 구성

- 김동현
- 조창일
- 이수진
# ⚙️ 개발 환경
- **언어** : C#
- **엔진**: Unity 2022.3.17f1
- **IDE**: Visual Studio 2022
- **프레임워크**: .NET 6.0
